# Udonthani Expo 2026

เว็บหน้าเดียวแบบง่าย ๆ สำหรับเปิดดูผ่าน VS Code Live Server

## วิธีเปิดดู
1. เปิดโฟลเดอร์นี้ใน VS Code
2. คลิกขวาที่ไฟล์ index.html แล้วเลือก "Open with Live Server"
3. ถ้าไม่มีตัวเลือกนี้ ให้ติดตั้ง extension ชื่อ Live Server

## วิธีอัปขึ้น GitHub
```bash
git init
git add .
git commit -m "Create one-page expo website"
git branch -M main
git remote add origin <URL ของ GitHub repo>
git push -u origin main
```

## จุดที่ควรแก้ไข
- ใส่ข้อความใหม่ตรงข้อความในไฟล์ index.html
- เปลี่ยนสีและรูปแบบในไฟล์ styles.css
- แก้การแปลภาษาในไฟล์ script.js
